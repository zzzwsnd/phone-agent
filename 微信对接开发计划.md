# PushPlus 微信推送开发计划

## Context

当前 `infra/wechat_push.py` 是占位实现，仅记录日志。需要替换为 PushPlus 真实推送，访客登记后自动通知保安微信。

PushPlus：关注公众号 → 注册 → 拿 Token → HTTP POST，无需企业认证，个人使用，免费额度 200 次/天。

## 需要用户提供

- **PUSHPLUS_TOKEN** — 在 pushplus.plus 关注公众号注册后获取

## 实现步骤

### 1. 添加环境变量配置

**文件**: `config/livekit_config.py`

```python
# PushPlus推送
PUSHPLUS_TOKEN = os.getenv("PUSHPLUS_TOKEN", "")
```

### 2. 添加 httpx 依赖

**文件**: `requirements.txt`

添加 `httpx`（异步 HTTP 客户端，比 aiohttp 轻量）

### 3. 替换 wechat_push.py 占位实现

**文件**: `infra/wechat_push.py`

- POST `http://www.pushplus.plus/send`，参数 `token` + `title` + `content`（markdown）+ `template=markdown`
- 保留 `format_wechat_message` 改为 markdown 格式
- 推送失败记录日志但不抛异常（不影响主流程）
- 添加重试（1次，间隔1秒）

### 4. 更新 .env.local

```
# PushPlus推送
PUSHPLUS_TOKEN=
```

## 涉及文件

| 文件 | 操作 |
|------|------|
| `config/livekit_config.py` | 添加 1 个环境变量 |
| `infra/wechat_push.py` | 替换占位为 PushPlus 推送 |
| `requirements.txt` | 添加 httpx |
| `.env.local` | 添加配置项 |

## 验证方式

```python
import asyncio
from infra.wechat_push import push_visitor_to_security
record = {"license_plate": "沪A12345", "visiting_company": "测试公司", "visitor_name": "张三", "caller_number": "13800138000", "purpose": "面试"}
asyncio.run(push_visitor_to_security(record))
```

确认微信收到 PushPlus 推送消息。
